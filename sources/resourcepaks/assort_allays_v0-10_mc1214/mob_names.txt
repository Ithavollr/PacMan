Hello!
This text sheet is for anyone looking to rename a mob into a specific skin variant! 
The first name will be the original texture of the mob followed by the names of the variants! 

- If any of the names has a "|" in it, the variant can be named in more then one way.
- If the text has *INSERT*, any generic colored Allay or texture can be used on top of the variant i.e. Red Lush Allay
    - All colored Allays with a * can be used for the *INSERT COLOR NAME* options
